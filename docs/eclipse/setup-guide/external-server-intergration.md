# External Server Integrations

With Eclipse, you can now link external services that contain your music media files stored on your hosted media servers.

## How to link your media server?

In Eclipse, navigate to the Settings page and select Cloud Storage, then tap on Add Connection. Select one of the available server types you would like to integrate and sign into your account. Once authenticated, your media will appear in the Library section under the Servers tab.
