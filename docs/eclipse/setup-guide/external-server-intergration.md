# External Server Integrations

You can now link your hosted media servers directly to Eclipse to stream your entire collection from anywhere.

## How to link your media server?

1. Navigate to the **Settings** page.
2. Select **Cloud Storage** and tap **Add Connection**.
3. Choose your server type (Plex, Jellyfin, WebDAV, etc.) and sign in.
4. Once authenticated, your media will appear in the **Library** section under the **Servers** tab.
