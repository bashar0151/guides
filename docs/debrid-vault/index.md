# Debrid Vault
![Debrid Vault Logo](/img/debrid vault/image-1765930838662.png)
## What Is It?
Debrid Vault (DV) is a feature-rich media manager that integrates with multiple of your debrid cloud providers [(Real-Debrid and TorBox)](https://bashar.wiki/Debrid). DV acts as a central location to manage all your media (movies, shows, music, comics, audiobooks, books and games) in one place. It doesn't stop there, DV lets you search, stream and organise your entire library and more. See some of the features below!

## What Features Does it Have?
- Built-in media player
- Playlist creation custom shareable playlists!
- Meta date scraper from Itunes, Spotify and Musicbrians!
- Search for music from various sources 
- Full Screen Player
- Miniplayer
- Tab Navigation: The Tab bar to switch between pages (search, hub, library) 
- Bottom Navigation search
- Set your Favourite songs
- Library Caching
- Multi-select Actions: You can select multiple songs to perform actions such as Add to Playlist, Add to Queue, and Download (for downloading files to local storage). 
- Stats Page: Review your listening stats
